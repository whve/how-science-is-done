# how-science-is-done
实验室里的条件差,调试了半个小时才正常,本来满肚子脾气,现在也是毫无脾气.  
实验室是一个十几人的大实验室,出路不好,学生文章少,主线做神经退行性疾病研究,从蛋白到细胞,到动物实验,再到生物信息学,整个流程到有所涉猎却无疑专精,依我所见,突破大概在500年后吧.  
我是本科老师推荐过来的,我也很珍惜,一开始我是毫无怨言,想好好做,为以后的发展铺平道路.  
可能是我本科老师把我夸得太厉害了吧,这里的老师认为我有大数据处理经验,我没法反驳,当他交代我一个nc课题的时候,还不忘补一句这个适合你.  
真的是适合吗?我不知道.我跌跌撞撞做到2018年5月,又开始怀疑自己.本来的我认为自己能变得像故事里的英雄那样优秀.  
好像有点跑题了,毕竟好久不曾记录自己的想法.  
无论如何,还是在这里记录下来,毕竟,在校一日,身不由己.
